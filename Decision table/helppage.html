How to Use the Decision Table Builder

The Decision Table Builder helps you create decision tables with custom conditions and actions. Follow these steps:

  1. Enter a title for your decision table in the Table Title field.
  2. Select the number of columns for your decision table using the dropdown menu.
  3. Click the Create Table button to generate your decision table.
  4. Add conditions by clicking Add Condition and entering a condition name.
  5. Add actions by clicking Add Action and entering an action name.
  6. Populate the table by selecting Y (Yes) or N (No) in each cell.

Once complete, your decision table will be ready for review!
